# Product Spec

## Product
Basic single-page frontend web app.

## Current Behavior
- Shows title: "Basic Frontend Webapp"
- Displays status message
- Button updates message with current local time when clicked

## Functional Requirements
- App loads without build step.
- Works on modern mobile and desktop browsers.
- Deployed and accessible on Vercel.

## Non-Goals (for now)
- Backend/API integration
- Authentication
- Complex state management

## Next Feature Areas
- Better visual branding/theme
- Small component additions (cards, form, nav)
- Accessibility pass (labels, focus, contrast)
